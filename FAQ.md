# FAQ

**Q: What sites are supported?**  
A: Any publicly accessible URL with readable text content.

**Q: How fast is WordIndexer?**  
A: Speed depends on site complexity but is highly optimized for multi-threading.

**Q: Does it handle non-English words?**  
A: Yes, WordIndexer supports UTF-8 text for global languages.
